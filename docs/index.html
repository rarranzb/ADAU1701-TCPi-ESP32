ESP32 · ADAU1701 · SigmaStudio

ADAU1701-TCPi-ESP32

WiFi bridge between SigmaStudio and the ADAU1701 DSP

  • Replaces ICP1 / ICP3 / ICP5 USB dongles
  • Hardware safeload — no pops or clicks
  • Web interface for WiFi and pin configuration
  • EEPROM write support for selfboot
Web installer requires Chrome or Edge on desktop. Use the command line method instead.
Requirements: Chrome or Edge browser · USB cable · ESP32 connected to PC

After flashing

  1. The ESP32 will create a WiFi network called ADAU1701-ESP32 (password: adau1701)
  2. Connect your phone or PC to that network
  3. Open http://192.168.4.1 and enter your WiFi credentials
  4. The ESP32 reboots and connects to your network
  5. In SigmaStudio: USBi → TCP/IP → enter the ESP32's IP and port 8086

Command line install

Linux / macOS / Windows (WSL): 

curl -fsSL https://raw.githubusercontent.com/rarranzb/ADAU1701-TCPi-ESP32/main/install.sh | bash

Manual flash

pip install esptool
esptool.py --chip esp32 --baud 460800 write_flash \
  0x1000  bootloader.bin \
  0x8000  partitions.bin \
  0x10000 firmware.bin

Download the .bin files from the latest release.
